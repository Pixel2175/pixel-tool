# Pixel Tool - 0.1V

#### Welcome to <b>Pixel Tool</b>, the program designed to enhance the performance of Windows 10 and 11 operating systems, clean and speed up the system, and activating Windows.

## features

> Reduce CPU processes.

> Activate Windows.

> Clean unnecessary files.

> Download all Runtime files.

> Uninstall unnecessary Windows programs.


## compile

> `WARNING`, in order to do this you must have python and git installed<br>

######

- `git clone https://github.com/Pixel2175/pixel-tool`
- `cd pixel-tool`
- `pip install -r requirements`
- `python -m PyInstaller --onefile main.py`

######

after this the compiled executable can be can be found in `/dist/main.exe`
